# 电商产品 — 分行业知识

## 关键指标
- **转化率**—— 访客 → 购买，行业均值 2–3%
- **AOV（客单价）**—— 用追加销售、套餐、包邮门槛来提升
- **购物车放弃率**—— 行业均值 70%，目标 < 55%
- **复购率**—— 90 天内再次购买的客户占比
- **ROAS（广告支出回报率）**—— 用于付费获客渠道

## 电商专属的 PRD 考量
- **商品目录**—— SKU 管理、变体（尺寸/颜色）、库存追踪
- **购物车**—— 跨会话持久保存，匿名 → 登录后合并
- **结算**—— 优先单页结算，必须支持游客结算（别强制注册）
- **支付**—— 至少 Stripe/PayPal，移动端支持 Apple Pay/Google Pay
- **物流**—— 实时运费计算、多承运商、包邮门槛
- **税费**—— 按辖区计算（用 Stripe Tax / TaxJar）
- **退货/退款**—— 自助退货申请、自动退款处理
- **订单追踪**—— 实时状态更新、邮件/短信通知

## 电商架构模式
- **购物车服务**—— 与订单服务分离，处理匿名 + 已登录
- **库存管理**—— 用乐观锁防止超卖
- **搜索**—— 带分面过滤（价格区间、颜色、尺寸、评分）
- **图片 CDN**—— 响应式图、WebP/AVIF、懒加载、可放大
- **推荐引擎**—— "买了又买"、"经常一起购买"

## 电商 UX 要求
- **商品页**—— 主图画廊、变体选择器、价格+省钱、信任信号（评价、退货政策）
- **加入购物车**—— 不跳页、抽屉/弹窗确认、"继续购物"选项
- **购物车**—— 改数量、删商品、优惠码输入、运费预估
- **结算**—— 物流 → 支付 → 复核 → 确认，带进度指示，保存地址供下次用
- **订单确认**—— 订单号、预计送达、商品摘要、邮件确认
- **移动端**—— 底部吸顶"加入购物车"按钮、可滑动图库、一键支付
